# TRUTHMODE Planner on Android

This app is now prepared as an installable Android PWA.

## Install on your phone

1. Publish the `docs` folder with GitHub Pages or another HTTPS web host.
2. Open the published link in Chrome on Android.
3. Tap **Install App** if the button appears, or use Chrome menu > **Add to Home screen**.
4. Open **TRUTHMODE** from your Android home screen.

The app works offline after the first successful load. Planner data is stored locally on the phone in browser storage, so use **Export** inside the app to back up your planner.

## Play Store Android app later

If you want a real `.apk` or Play Store app, the same PWA can be wrapped with Bubblewrap/Trusted Web Activity. That requires an HTTPS published URL, Android Studio/JDK, and signing keys.
